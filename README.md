# My Todo List App - Milestone 1

## Project Overview
A mobile-friendly todo list application for Object Oriented Programming course.

## Features
- ✅ Responsive mobile-friendly design
- ✅ "My Todo List" title
- ✅ Hardcoded todo list (Buy milk, Buy bread, Buy eggs)
- ✅ "Add New Todo" button
- ✅ Real-time clock display
- ✅ Modern gradient UI

## How to Run
Open `index.html` in any modern browser.
